Для создания учетной записи необходимо раскомментировать роуты регистрации в файле web.php

Принцип работы - мы можем добавлять, изменять, удалять сущности item, дополнительно к каждой сущности также записывается в отдельную таблицу история ее изменений, которая выводится на странице изменения сущности.
